## [1.0.0] - 2025-02-15

### Added
- Initial release of Skills Builder
- `check` command to validate skill structure
- `create` command to generate new skills
- `init` command to initialize directory as skill
- `update` command to modify existing skills
- Support for multiple output formats (text, json, markdown)
- Internationalization support via polyglot.json
- Best practices validation for skill projects
